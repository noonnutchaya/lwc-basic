# LWC Basic

Lightning Web Components を基礎から学習していきましょう。

## Common Values

- **"習うより慣れよ"**

  理解してから書くのではなく、書いてから理解しましょう。

- **"とにかく写経しよう"**

  お手本を見ながら書き写すことが上達の近道です。コピペは禁止。

- **"ログとお友達になろう"**

  エラーが発生しても焦らないでください。落ち着いてログを 1 行ずつ読みましょう。
